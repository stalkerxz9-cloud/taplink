Создание страниц Taplink для шоурумов 
auto.ae


Срок выполнения: 30 дней



1. Цель

Создать для каждого шоурума из каталога 
https://auto.ae/showrooms/all/
 отдельную страницу на платформе Taplink.
Страницы должны быть доступны по ссылке формата:
https://taplink.cc/SHOWROOMNAME


Всего необходимо обработать около 600 шоурумов.



2. Источник данных

Все данные берутся непосредственно с сайта 
auto.ae
:

· Исполнитель заходит в каталог 
https://auto.ae/showrooms/all/
, выбирает шоурум, переходит на его страницу.
· На странице шоурума находятся ссылки:
  1. Сам шоурум  URL страницы шоурума (из адресной строки).
  2. Авто шоурума  ссылка на каталог автомобилей.
  3. Аренда авто  ссылка на раздел аренды.
  4. Автомобильные номера шоурума  ссылка на раздел с номерами.
  5. Проданные автомобили шоурума  ссылка на раздел проданных авто.
· Дополнительно: собрать 510 фотографий топовых автомобилей из каталога шоурума (первые авто в списке) для размещения в галерее.



3. Этапы выполнения

3.1. Сбор ссылок и фотографий с каждого шоурума

· Перейти по каждому шоуруму в каталоге.
· Скопировать:
  · Название шоурума.
  · URL страницы шоурума (основная ссылка).
  · URL раздела «авто» (каталог автомобилей).
  · URL раздела «аренда».
  · URL раздела автомобильных номеров.
  · URL раздела проданных автомобилей.
  · 510 фотографий топовых автомобилей (скачать или сохранить ссылки на изображения из каталога шоурума).
· Зафиксировать в рабочей таблице (Google Sheets или Excel) для последующего создания аккаунтов.

3.2. Создание аккаунтов Taplink

· Для каждого шоурума зарегистрировать отдельный аккаунт на Taplink.
· В качестве имени пользователя (username) использовать транслитерированное или сокращённое название шоурума, чтобы итоговая ссылка имела вид 
taplink.cc/название_шоурума
.
· Обеспечить уникальность username (при конфликте добавить суффикс: город, цифру и т.п.).
· При необходимости использовать разные email-адреса и прокси для обхода ограничений платформы.

3.3. Настройка профиля Taplink

· Название профиля: название шоурума (как на 
auto.ae
).
· Фото профиля: логотип шоурума (берётся со страницы шоурума).
· Фоновое изображение: презентабельное фото (автомобиль, шоурум), взятое со страницы шоурума или из его объявлений.
· Кнопка "Связаться": форма обратной связи или ссылка на WhatsApp (если номер указан на странице шоурума).
· Био (описание): на основе данных со страницы шоурума составить описание с использованием ИИ.
· Галерея автомобилей: добавить блок с каруселью из 510 фотографий топовых автомобилей шоурума (собранных на этапе 3.1).
· Добавление ссылок в строгом порядке (с помощью блоков Taplink  кнопки, виджеты):
  1. Сам шоурум  ссылка на страницу шоурума на 
auto.ae
.
  2. Авто шоурума  каталог автомобилей.
  3. Аренда авто  раздел аренды.
  4. Автомобильные номера  раздел с номерами.
  5. Проданные авто  раздел проданных автомобилей.

Все ссылки должны быть рабочими и вести на соответствующие разделы. Даже если разделы пустые, ссылки добавляются.

3.4. Формирование итоговых ссылок

· Каждая страница должна быть доступна по единому формату:
    
https://taplink.cc/название_шоурума


3.5. Контроль качества

· Проверить открытие каждой страницы Taplink.
· Проверить каждую добавленную ссылку (чтобы не было битых ссылок, редиректов на неверные страницы).
· Проверить отображение галереи (510 фото, корректная загрузка).
· Убедиться, что для одного шоурума не создано несколько аккаунтов.



4. Требования к исполнителю

· Опыт работы с Taplink (регистрация, массовое создание аккаунтов, настройка блоков/кнопок, добавление галереи).
· Внимательность к деталям: все ссылки должны соответствовать типу (шоурум, авто, аренда, номера, проданные).
· Предоставление промежуточных отчётов (каждый день) и итогового результата.



5. Результат и критерии приемки

· 600+ страниц Taplink созданы (по числу шоурумов).
· Каждая страница содержит оформление (логотип, фон, био), галерею из 510 фото топовых автомобилей и все указанные ссылки в правильном порядке.
· Все ссылки рабочие и соответствуют категориям.
· Галерея корректно отображается (510 фото, не битые).
· URL страниц имеют вид 
https://taplink.cc/название_шоурума
.
· Итоговый отчёт в формате Google Sheets / Excel со столбцами:
  · Название шоурума
  · Ссылка на страницу Taplink
  · Ссылка на шоурум (
auto.ae
)
  · Статус (успешно / ошибка с пояснением)
  · Примечание (отсутствующие ссылки, особенности)
