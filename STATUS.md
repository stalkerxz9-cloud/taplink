# Статус проекта: Автоматизация Taplink (Auto.ae)

## 🎯 Общая цель

Автоматическое создание и настройка более 600 профилей Taplink для автомобильных шоурумов, спарсенных с сайта Auto.ae.

## ✅ Что уже сделано

1. **Парсер (`1_parser.js`):** Сбор данных о шоурумах.
    * **NEW:** Оптимизировано управление браузером (переиспользование инстанса для всех страниц).
    * Добавлена пагинация (50 страниц) для полного сбора 600+ объектов.

2. **Генератор BIO (`2_bio_generator.js`):** Создание уникальных описаний для каждого профиля.
3. **Бота регистрации (`3_taplink_bot.js`):**
    * **NEW:** Реализован План А (1secmail.cc) с кнопкой Delete для моментальной смены адреса.
    * Постоянная вкладка почты (mPage) для максимальной скорости внутри батча.
    * Регистрация через временную почту с авто-подтверждением кодом.
    * RuCaptcha интегрирована для обхода защиты.
    * Установка пароля и базовых настроек.
4. **Оформление профилей (`Design`):**
    * **NEW:** Автоматическая установка аватара (логотипа) из локальных файлов.
    * **NEW:** Настройка BIO (AI) и блока «Текст».
    * **NEW:** Добавление Галереи (5-10 фото) в режиме карусели.
    * **NEW:** Добавление 5 обязательных кнопок-ссылок в строгом порядке.
    * **NEW:** Авто-фон из первого фото объявлений.
5. **Стабилизация системы:** 
    * **NEW:** Логирование в файл `data/bot_log.txt` для предотвращения вылетов VS Code.
    * **NEW:** Поддержка режима «Только оформление» для уже созданных аккаунтов.
    * **NEW:** Групповое управление браузером (один инстанс на 10 шоурумов).
    * Скрипт `cleanup.ps1` для принудительной очистки Chromium.

## 🛠 Текущие настройки

- **Пароль для всех аккаунтов:** `SecureShowroom#2024`
- **Файл данных:** `data/showrooms_data.json`
- **RuCaptcha Key:** Установлен в `config.json`.

## 📋 Следующие шаги (Запуск)

1. **Запустить парсер:** `node 1_parser.js` (сбор всех шоурумов).
2. **Запустить генератор BIO:** `node 2_bio_generator.js`.
3. **Запустить бота:** `node 3_taplink_bot.js` (регистрация).
